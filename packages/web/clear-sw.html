Why is this needed?
The service worker is caching the old HTML with the Tailwind CDN script. This tool will clear all caches and unregister the service worker.

Manual Steps (Alternative)

  1. Open Developer Tools (F12)
  2. Go to Application tab
  3. Click Service Workers in sidebar
  4. Click Unregister
  5. Click Storage in sidebar
  6. Click Clear site data button
  7. Refresh the page
